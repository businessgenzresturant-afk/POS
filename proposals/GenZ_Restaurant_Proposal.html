4. Features Overview

4.1 Point of Sale (POS) Terminal

  • Visual table layout with status indicators
  • Quick item search by name or category
  • Modify orders (add/remove items, special instructions)
  • Apply discounts (percentage or fixed amount)
  • Split bills by seat or by amount
  • Multiple payment methods (Cash, Card, UPI, Wallet)
  • GST-compliant invoices with automatic tax calculation

4.2 Menu Management

  • 170+ pre-loaded menu items from GenZ Restaurant
  • Categories: Tandoor, Chinese, Main Course, Breads, Biryani, Beverages, etc.
  • Upload images for each dish
  • Toggle availability in real-time
  • Set pricing with GST slabs (5%, 12%, 18%)

4.3 Kitchen Order Ticket (KOT) System

  • Real-time order display in kitchen
  • Color-coded status: Pending → Preparing → Ready → Served
  • Timer display showing order age
  • Multiple kitchen station support

4.4 Dashboard & Reports

  • Daily sales summary
  • Payment mode breakdown (Cash/Card/UPI/Wallet)
  • Top-selling items (by quantity and revenue)
  • Hourly sales trends
  • Export to PDF/Excel for accounting
